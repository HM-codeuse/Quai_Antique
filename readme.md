# Quai Antique

Le Quai Antique est un restaurant de Chambéry qui propose la cuisine traditionnelle de Savoie du chef Arnaud Michant. 

## Environnement de développement 

### Pré-requis
-   PHP 8.0.25
-   Composer 2.5.4
-   Symfony CLI
-   nodejs et npm

### Lancer l'environnement de développement 

``bash
symfony serve -d
``
## Lancer des tests

``bash
php bin/phpunit
``
